# Fork Checker

このツールはSymbolのノードのフォーク状況をチェックすることができるツールです。

使い方は調べたいノードのURLを入力し、チェックを押すだけです。

チェックはノードの最新ブロックから5ブロックほど前より始まります。

また、チェックするブロック高を指定することもできます。

ツールは[こちら](https://symbol-blockchain-community.github.io/fork-checker)
